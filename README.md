# ARAS BAR - Nagold

Modern, responsive website for ARAS BAR (Shisha, Bar, Lounge) in Nagold.

## Features
- **Dynamic Opening Hours:** Automatically displays whether the bar is currently open or closed based on local time.
- **Premium Design:** Dark mode UI with gold accents, smooth animations (via Framer Motion), and a frosted-glass navigation.
- **Responsive Layout:** Fully optimized for mobile and desktop screens.
- **Google Maps Integration:** Embedded map for easy navigation.
- **SEO Ready:** Fast performance with Next.js App Router and Next/Image.

## Tech Stack
- Next.js (App Router)
- React
- Framer Motion
- Lucide React (Icons)
- Vanilla CSS

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
